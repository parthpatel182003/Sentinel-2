# Data Policy

Do not store full raw satellite scenes in GitHub.

Use this repository for:

- code
- notebooks
- sample outputs
- metadata schemas
- documentation

Store large raw and intermediate files in Google Drive.

Recommended Drive directories:

- `data/raw/sentinel2/`
- `data/raw/labels/`
- `data/interim/`
- `data/processed/`

If you want a public demo in this repository, add:

- a tiny sample chip
- one sample mask
- one sample overlay
- a short manifest CSV
